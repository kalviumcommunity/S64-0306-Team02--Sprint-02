## RuralEduTrack 📚
Smart Attendance and Academic Progress Monitoring App for Rural Coaching Centers
Overview

Rural coaching centers often struggle to consistently track student attendance and academic progress due to the lack of digital tools. Many institutions still rely on manual registers, which can lead to errors, lost records, and difficulty in analyzing student performance.

RuralEduTrack is a mobile application built using Flutter that provides a simple and accessible solution for teachers to manage student records, track attendance, and monitor academic progress. The app aims to digitize classroom management while remaining easy to use even in low-resource environments.

## Problem Statement

Rural coaching centers find it difficult to track student attendance and academic progress consistently due to limited digital systems. This project aims to develop a simple, reliable, and accessible mobile application that helps teachers efficiently manage attendance and student performance.

## Project Goals

Digitize attendance tracking

Simplify student record management

## Key Features

## 1. Teacher Login

Secure login system for teachers to access the application and manage student records.

## 2. Student Management

Teachers can:

Add new students

Edit student details

Delete student records

View a list of enrolled students

## 3. Attendance Tracking

Teachers can mark students as:

Present

Absent

Attendance records are stored and used to calculate attendance percentages.

## 4. Academic Progress Tracking

Teachers can record:

Test scores

Assignment marks

Subject-wise performance

This helps track student improvement over time.

## 5. Dashboard Analytics

The app includes a dashboard that shows:

Attendance statistics

Student performance charts

Overall class progress

## 6. Report Generation

Teachers can generate reports including:

Monthly attendance reports

Individual student progress reports
